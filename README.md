# ProconsApp

Procons app is an application that manage pros and cons to help answer a question. It consumes npm procons library.

## Development server

To start a local development server, run:

```bash
npm run start
```

## Lint errors

```bash
npm run lint
```

## Build application

```bash
npm run build
```

## Running unit tests

```bash
npm run test
```

or test headless

```bash
npm run test:chrome-headless
```

or code coverage

```bash
npm run test:code-coverage
```
